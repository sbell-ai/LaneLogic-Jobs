- `FULL_MD_RE = /^\[(.+?)\]\((.+?)\)$/`
- `EMBED_MD_RE = /\[(.+?)\]\((.+?)\)/`

- `stripMdText(s)` returns capture group 1
- `stripMdUrl(s)` returns capture group 2
- Always `.trim()` input first
- Try `FULL_MD_RE` first, then fall back to `EMBED_MD_RE`, else return trimmed original

- `domain` → `stripMdText(domain)` then normalize to hostname (remove `http(s)://` and trailing path)
- `website`, `primarySource`, `secondarySource` → `stripMdUrl(...)`
