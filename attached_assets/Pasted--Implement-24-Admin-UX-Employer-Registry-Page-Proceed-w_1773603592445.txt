## Step 1 — Create shared admin-session guard helper

Create: **`server/middleware/requireAdminSession.ts`**

Requirements:

- Export `requireAdminSession(req, res): boolean`
- Must require both:
    - `req.isAuthenticated()` is truthy
    - `req.user.role === "admin"`
- If not authenticated: return **401**
- If authenticated but not admin: return **403**
- Response body should be JSON with `{ ok:false, error: "...", message: "..." }`
- Must not log secrets

In **`server/routes.ts`**, add two routes that call registry functions directly (no HTTP forwarding, no secrets):

1) **GET** `"/api/admin/registry/employers"`

- Guard with `requireAdminSession`
- `environment = NODE_ENV === "production" ? "prod" : "staging"`

- On sync success: invalidate query key `["/api/admin/registry/employers"]`

UI requirements:

- Heading: “Employer Registry”
- Buttons:
    - **Sync now** (shows spinner while pending, then refreshes table)

    2) Domain

    5) Accepted evidence count
    
    6) Accepted source types (chips/badges)
    
    7) Primary Source + Secondary Source (clickable external links, omit if empty)
